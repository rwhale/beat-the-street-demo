# Beat the Street

A personal quantitative trading project — building one breakout strategy, testing it honestly across the years it was tuned on and a year it had never seen, and now proving it (or disproving it) in a live paper-trading run.

## What the page shows

- **The hypothesis** — what the strategy bets on, in plain English.
- **How it was tested** — a strict split between the build years (2019–2023) and a held-out year (2024) used exactly once and never reused, plus a 1,000-run shuffle (Monte Carlo) test.
- **The verdict so far** — the honest reading, warts included: negative across the build years, strongly positive in the one held-out year, slightly positive combined.
- **The live test** — a live paper-trading run (real prices, pretend money) that will give the real answer.

## A note on honesty

This page deliberately shows the weak results alongside the strong ones. A slightly-positive past is a reason to run the live test, not a reason to believe the strategy works. That conclusion is still open.

## Disclaimer

Beat the Street is a personal research project. Every result shown comes from backtests and live paper trading — no real money is involved. Nothing here is investment advice or an offer to manage anyone's money.

The trading system itself (code, data, credentials) is kept private. This repository contains only the read-only showcase page.
